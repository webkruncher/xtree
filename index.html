Welcome to my binary tree project.

This is a work in progress.

This project is mainly designed as a learning tool. The idea is to build a set of C++ objects that can be used to create any type of binary tree. There are functions built into the tree bases that provide hooks so that the user code can display the nodes as they are inserted and deleted. This makes it so that a coder can modify the tree code, and see the impact on the screen.

Status

The basics of a binary tree are implemented, and parameterized in a way that makes it possible to create either a map or a set class, somewhat analogous to an STL map and set.

The tester.cpp source is used to test the add and delete throughput against the stl set.
